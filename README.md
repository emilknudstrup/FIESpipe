# FIESpipe
A pipeline to extract CCFs, BISs, activity indicators, RVs etc. from FIES spectra



### Installation
`cd /path/to`

`git clone https://github.com/emilknudstrup/FIESpipe.git`

`cd /path/to/FIESpipe`

`python -m pip install .`
